1. DISC Model Overview (Professional Context)
DISC measures observable workplace behavior, not personality or ability.
Dimension
	Core Motivation
	Workplace Focus
	Dominance (D)
	Results, control
	Decision-making, leadership
	Influence (I)
	Social approval
	Communication, persuasion
	Steadiness (S)
	Stability, harmony
	Teamwork, support
	Conscientiousness (C)
	Accuracy, quality
	Structure, analysis
	DISC answers one question:
👉 How does this person behave in a work environment?
Not intelligence. Not values. Not emotional health.
________________


2. Professional DISC Questionnaire Design
There are two standard formats:
Format A — Forced Choice (Most Professional)
Format B — Likert Scale (Simpler but less accurate)
Professional tools prefer Forced Choice.
________________


3. Forced-Choice DISC Question Structure (Professional Standard)
Each question contains 4 statements, each mapped to D, I, S, C.
Candidate must select:
• MOST like me
• LEAST like me
________________


Example DISC Question (Professional)
Question 1: In a work setting, I tend to:
A. Take charge of situations quickly
B. Encourage and motivate others
C. Support team harmony
D. Focus on accuracy and details
Mapping key:
A = D
B = I
C = S
D = C


Candidate selects:
MOST: A
LEAST: C
Scoring:
D += 1
S -= 1

6. DISC Scoring Algorithm (Professional Standard)
Initialize:
D = 0
I = 0
S = 0
C = 0


For each question:
Most_choice → add +1
Least_choice → subtract -1


Example:
MOST = D → D += 1
LEAST = S → S -= 1


Final raw scores example:
D = 8
I = 4
S = -2
C = 6


Normalize to positive scale:
Add constant (example: +10 to all)


Final:
D = 18
I = 14
S = 8
C = 16
